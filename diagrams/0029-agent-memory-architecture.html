Cloud layer — conversation turns; what agents said; cross-session continuity. External dependency. Auto-captured.
Local layer — agent identity, skills, MEMORY.md, daily journals. Fully local. Sovereign. Auto-indexed from files.
Learnings layer — operational intelligence written explicitly by agents. Shared across all agents. On-demand query only.
No sync between layers by design. Each layer captures a distinct memory type; overlap is minimal in practice.
